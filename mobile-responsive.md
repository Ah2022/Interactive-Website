## Notes on Mobile Responsiveness:
- Single column layout for all content
- Hamburger menu for navigation
- Featured projects stack vertically instead of horizontal scroll
- MDR Game Project shows one view at a time with toggle to switch
- Skills display in 2-column grid instead of 5-column
- Form fields expand to full width on contact page
- Animations scale appropriately for smaller screens
- Touch interactions replace hover effects
- Maintains all core functionality with adapted layout
