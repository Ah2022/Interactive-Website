## Notes:
- Filter tabs at top allow sorting projects by technology
- Projects displayed in responsive grid layout (2x2 on desktop)
- Each project card contains Rive animation, title, description, tags, and CTA
- MDR Game Project gets special treatment with split identity UI
- Toggle switch allows switching between Innie and Outie views
- Each view has its own animation, description, and styling
- Animations activate on hover/click
- Background gradient shifts subtly based on active filter
